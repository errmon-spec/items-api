## Desenvolvimento local

Este projeto oferece suporte à contêineres de desenvolvimento (dev container).

### Docker compose

```bash
docker compose -f .devcontainer/docker-compose.yml run --rm runner
```

### VS Code

```bash
devcontainer open .
```
